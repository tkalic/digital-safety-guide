# Contributing

Contributions are welcome. This guide is meant to stay practical and current — both of which require ongoing input.

---

## What's Useful

- **Real-world examples** — specific, documented cases that illustrate a point better than a general statement
- **Outdated information** — regulations change, tools get deprecated, statistics age. Flag anything that needs updating
- **Missing topics** — if something relevant isn't covered, open an issue
- **Corrections** — if something is factually wrong, please raise it

---

## What Doesn't Fit

- Generic advice that's already covered
- Tool recommendations without context for why they're better than what's listed
- Content that's opinion-only without supporting reasoning

---

## How to Contribute

1. Fork the repository
2. Make your changes in a new branch
3. Open a pull request with a clear description of what you changed and why

For larger changes or new sections, open an issue first to discuss the direction before writing.

---

## Style Notes

- Write for someone with no technical background — explain terms the first time you use them
- Prefer concrete examples over abstract principles
- Short paragraphs, direct language
- No marketing language ("empower", "leverage", "synergy")
