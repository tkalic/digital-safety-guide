# Checklists

Practical, actionable. Work through these once — then make them habits.

---

## Personal Data Privacy

- [ ] Using a password manager with unique passwords per service
- [ ] Two-factor authentication enabled on email, banking, and social media
- [ ] Checked [haveibeenpwned.com](https://haveibeenpwned.com) for breached accounts
- [ ] Reviewed app permissions on phone — revoked anything unnecessary
- [ ] Privacy settings reviewed on social media platforms
- [ ] Using HTTPS — browser shows padlock icon on sensitive sites
- [ ] VPN installed for use on public Wi-Fi
- [ ] Old accounts deleted or deactivated

---

## Safe Browsing & Communication

- [ ] Using a privacy-focused browser (Firefox or Brave)
- [ ] uBlock Origin or equivalent ad/tracker blocker installed
- [ ] Suspicious links hover-checked before clicking
- [ ] Sensitive conversations via encrypted messaging (Signal, not SMS)
- [ ] Not logging into sensitive accounts on public/shared devices

---

## Recognizing Threats

- [ ] Know what phishing looks like — urgency, suspicious sender, credential requests
- [ ] Don't install cracked software or use unofficial download sources
- [ ] Software and OS kept up to date
- [ ] Know how to verify someone's identity before acting on requests involving access

---

## Responsible Social Media Use

- [ ] Reading articles before sharing, not just headlines
- [ ] Cross-checking claims with a second source before treating as fact
- [ ] Not posting real-time location information
- [ ] Aware of what's publicly visible on your profiles
- [ ] Reporting harmful or misleading content rather than just scrolling past

---

## For Developers & Organizations

- [ ] User data collection limited to what's strictly necessary
- [ ] GDPR/CCPA compliance reviewed if handling EU or California user data
- [ ] AI models audited for bias before deployment
- [ ] Responsible disclosure policy in place for security vulnerabilities
- [ ] Hosting provider checked for renewable energy use
- [ ] Security training included in onboarding
